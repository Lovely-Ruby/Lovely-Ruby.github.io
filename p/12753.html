【前端18_JS】ES 6：箭头函数、This指针、处理错误、数据结构 Set、WeakSet、数据类型 Symbol、魔术字符

发表于2019-11-07|更新于2024-02-10|计算机语言
|字数总计:2.4k|阅读时长:10分钟|阅读量:

箭头函数

参数

  • 最简单的箭头函数：一个形参可以不用括号
  • 函数如果没有形参，需要加括号
  • 多参数也需要加括号

用法 & 问题

  • 箭头函数不能直接返回对象,他会当作代码块去处理，解决方法：在对象外 套上圆括号 即可
  • 函数如果没有返回值，打印的是 undefined
  • 把很多实参捏到一起，变成真正的数组，用 ...arr ，也叫做 rest
  • 和 map 映射结合,把数组中的每一项都加 1
  • 和 sort 结合，进行从小到大排序

与 this 的联系

用了箭头函数，那么 this 指向的就是对象，而不是谁调用就指谁了

再来一个例子慢慢理解一下

缺点

  • 不能使用箭头函数实例化对象了
  • 不能用 arguments 获取传来的参数了

处理错误

  • 一般由 try、catch、finally、throw 几个构成
  • try：尝试执行代码，这里遇到的第一个错误会带到 catch 中
  • catch：处理错误，可以把错误吞并掉，从而避免卡住浏览器
  • finally：无论 try 中出错与否，都会执行
  • throw：抛尸，比方说 throw new error("你输错了");

数据结构：set

这个数据结构的特点是：元素不会重复
它没有下标，没有所谓的 length 长度

规则

  • 重复元素的判断跟 === 差不多，不同的是 NaN，如果有两个 NaN，他会认成一个，也就是书面一样

长度

  • 用 set.size

增加元素

  • 用 set.add()

删元素

  • 用 set.delete() 删除元素，返回布尔值，表示成功与否

查找元素

  • 用 set.has(""); 查找元素，返回布尔值表示，true 为存在

遍历

遍历可以用如下的方法

  • 用 Array.from(set)
  • 结构解析[...set]
  • forEach 循环

清空

  • 用 set.clear(); 来清空

曲线救国实现 map 方法

数组去重

用 set 的特性啊，没有重复的元素
想得到么小伙子

求两个 set 元素的交集

关键，用过滤器 filter，他会保留返回值是 true 的

WeakSet

  • 只能添加 Object 的 set
  • WeakSet 不允许被循环
  • WeakSet 里面放节点-防止内存泄露-JC
  • 如果想添加个数组

数据类型 Symbol

ES 6 中又新添加了个数据类型

  • 他的值 独一无二
  • 建议放在对象中做属性名（因为它的特性–永远不会重复）
  • 用 Object.keys() 是拿不出来 Symbol 的，根据这个特性，可以伪装成私有属性，for in 循环更是白扯，啥都不输出

取值

它有自己的方法 Object.getOwnPropertySymbols(obj)

Symbol.for()

Q:有人就想问，Symbol 既然是独一无二的，真的就不能重用么？

引进新的方法：Symbol.for(); 只要描述相同的话，那么他俩就是相同的

取 ‘ 描述 ’

  • symbol.description 可以拿出属性
  • Symbol.keyFor(b) 只能拿出 Symbol.for("") 设定的属性

魔术字符

就是出现频率贼 JB 高，改起来贼 JB 费劲，一改就要改一大堆的的字符串就叫做魔术字符

官方说法：我们把 耦合性极强，复用性极差，出现频率极高 的字符串成为魔术字符

那解决的方法蛮多的，用个变量代替它不就好了。要改的话，就改变量的值就好了

深入理解 Object 和 Map

Object

  • 对象结构来源于：hash 结构
  • 对象的 key 是会自动 toString 的
  • 是 string - value 的组合

Map

  • map 的 key 值没有 toString，很自由，想放什么放什么
  • 是 value - value 的组合
