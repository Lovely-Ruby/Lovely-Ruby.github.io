简介

知识点补充

token：服务端登录成功后进行签发
哈希：对复杂的数据进行摘要，10G 的内容，如果改了1kb ，那么它的哈希值也会改变的

实现方式

流程：我们以登录过程为例子，登录成功后签发 token，访问接口时携带 token

签发 Token

可以通过 jsonwebtoken 这个模块去签发 token，参考 npm 上的模块说明，也可以参考 https://jwt.io/ 查看 token 的信息
